# Arduino Embedded Systems Project

## Overview
Led a team of five to design and implement three embedded systems applications using Arduino IDE and C++.

## Projects Included

### Task 1: Mini Weather Station
- DHT20 temperature and humidity sensor
- BMP280 pressure sensor
- OLED display for real-time readings
- Buzzer alert when temperature exceeds threshold
- Potentiometer to adjust temperature threshold

### Task 2: Smart Night-Light System
- Light-dependent resistor (LDR) for ambient light sensing
- Potentiometer to set light threshold
- LED automatically turns on when dark
- OLED display shows light level and threshold

### Task 3: Pedestrian Crossing Simulation
- Button-activated countdown timer
- Red LED for stop signal
- Buzzer alerts for crossing start and end
- Adjustable timing via potentiometer
- Flashing LED warning before returning to stop

## Technologies Used
- Arduino IDE
- C++
- DHT20 & BMP280 sensors
- SSD1306 OLED display
- LEDs, buzzer, potentiometer, LDR
